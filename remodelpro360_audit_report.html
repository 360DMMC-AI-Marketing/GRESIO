Overview

RemodelPro360 is a renovation/remodeling marketplace connecting homeowners with contractors. The platform enables homeowners to post projects, browse contractor portfolios, and manage bids — while contractors can showcase their work, manage availability, and bid on projects.

Key Strengths

  • Robust authentication — JWT + Google OAuth, role selection, profile management, password reset
  • Contractor features are fully functional — availability, bids, earnings, bid templates, notifications
  • 9 contractor profiles with rich data (GeoJSON service areas, specialties, ratings)
  • Clean response patterns with consistent JSON structure

Key Weaknesses

  • 4 endpoints crash with 500 — Reviews (x2) and Payments detail (x2) throw internal server errors
  • Email verification gates write operations — Cannot create projects, designs, or make changes until email is verified
  • Missing list endpoints — No index routes for bids, change-orders, contracts, milestones, or portfolio
  • Projects search is broken — GET /projects/search/by-status returns 400 "INVALID_STATUS" regardless of input
  • Messages missing conversation list — No way to list conversations, only per-project and unread-count

Final Verdict

RemodelPro360 is a solid marketplace platform for the renovation/remodeling industry with a strong authentication system and fully functional contractor features. The contractor experience is well-developed — availability management, bid templates, earnings tracking, and notifications are all working. The contractor discovery API returns rich profiles with GeoJSON service areas and specialties. However, the platform has 4 endpoints crashing with 500 errors (reviews and payment details), and the email verification gate blocks meaningful testing of write operations. The absence of list endpoints for bids, contracts, milestones, and change-orders limits the platform's operational completeness. Once the 500 errors are fixed, email verification is completed, and missing list endpoints are added, RemodelPro360 will be a strong competitor in the home renovation marketplace space. Overall maturity: BETA stage — Contractor features production ready, marketplace features need work.
